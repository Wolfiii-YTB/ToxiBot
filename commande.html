Commandes d'ajout /ajoutermot

La commande d'ajout sert a ajouter un mot a bannir dans votre serveur.

Commandes de liste /liste

Cette commande sert a voir tous les mots qui sont banni de votre serveur.

Commandes pour supprimer un mot /supprimer

Cette commande sert a supprimer un mot de votre liste des mots bannis.

Commandes pour ajouter un sondage /sondage

Cette commande permet de créer un sondage sur votre serveur, avec un maximum de cinq options. Une fois le sondage créé, ToxiBot ajoutera automatiquement les émojis numérotés "1", "2", etc., correspondant à chaque option.

Commandes pour créer des tickets /ticket

Cette commande permet aux membres de votre serveur de créer un ticket via une commande, avec la possibilité d’indiquer jusqu’à 3 motifs maximum. Les personnes ayant accès à ce ticket sont celles possédant le rôle "admin". (Cliquez ci-dessous pour obtenir une explication du module admin.) Explication du role admin
